重归博客

Table of Contents
  • END

转眼已经一年没写博客了，这一年也发生了很多事情，疫情，最难毕业季，所幸准备的还算充分，不至于颗粒无收。个人视频频道也在稳步前进，希望以后涨粉多多。

有段时间发现博客的网址挂了，查看后发现原来托管的 coding 平台删库跑路了，个人版 coding 全部删除。看来还是 github 靠谱些。于是这几天重新整合下把网页又托管在了 github 下，同时换了个新的 hugo 主题。

这一年看的电影和书都很少，倒是看了不少综艺，还真是惭愧呢。争取影评和读书笔记系列都能继续更下去。同时硬核知识开始变主力军了，整理下发一些自己碰到的各种坑。

那么，新的时期开始了。逐渐步入社会，成为打工人，也希望以后一帆风顺吧。
